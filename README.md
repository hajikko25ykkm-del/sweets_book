# sweets book
## サイト概要
### サイトテーマ
お菓子作りを最適化する、レシピ管理と在庫管理ができるコミュニティサイト
​
### テーマを選んだ理由
これまでは作るたびにレシピを検索していましたが、以前気に入ったレシピが見つからず、頭を悩ませることが多々ありました。また、お菓子作りは多くの道具を使いますが、レシピで指定された型と手持ちのサイズが合わないことも多く、そのたびに計算し直す手間や、仕上がりの間違いが発生するのも課題でした。</br>
さらに、材料の残量や期限を把握しきれず、作り始めてから「足りない」と気づいたり、逆に余らせてしまったりすることもあります。</br>
こうした二度手間やミスを防ぎ、大好きなレシピをすぐに確認出来たら便利だと考え、このテーマにしました。
​
### ターゲットユーザ
・いろいろなお菓子を作っていて材料管理が苦手な人</br>
・お気に入りのレシピを保存しておきたい人</br>
・お菓子つくりが好きだけど、時間がなくて次の機会が訪れずに材料を捨ててしまう人
​
### 主な利用シーン
・計画的、衝動的にお菓子を作りたくなり買い物をする時</br>
・作るレシピの確認したい時</br>
・忘れていた材料消費の時
